# IdiotBubble
Android 气泡弹框组件

借助PopupWindow实现的一个通用提示气泡组件

## ScreenShot
![PreView](./demo_show.png)

## Version
### 1.0 支持：
- 指定气泡颜色、方向、距离
- 指定阴影颜色、偏移
- 指定气泡圆角弧度
- 指定气泡距离
